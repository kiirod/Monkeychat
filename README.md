# About
Monkeychat is like Twitter's DM system, 'XChat', but for Monkeytype. This is a tool built for Monkeypost as well.

# Data Management
We don't take any data from you, including your personal information such as legal name, house address, credit card numbers, etc... Monkeychat is just a website where you can message people from Monkeypost. It'd be pointless to add tracking.

More information; [Privacy Policy page](https://pj.mtgoals.cc/home/app/policy)
